# Exercise 1 - Configuring a Basic Spring Application

## Objective

Configure a basic Spring application using Spring Core and XML-based bean configuration.

## Technologies Used

- Java 8
- Maven
- Spring Core (spring-context)

## Project Structure

- BookService
- BookRepository
- applicationContext.xml
- LibraryManagementApplication

## Steps Performed

1. Created a Maven project.
2. Added Spring Core dependency in pom.xml.
3. Created BookService and BookRepository classes.
4. Configured beans in applicationContext.xml.
5. Loaded Spring Application Context.
6. Retrieved beans from the container.
7. Executed the application.

## Output

Book Service Loaded

Book Repository Loaded

## Result

Successfully configured and executed a basic Spring application using Spring IoC Container and XML configuration.
