# Exercise 4: Creating Maven Project

## Objective

Create and execute a simple Maven Project using Maven build tools.

## Project Structure

- App.java
- pom.xml
- README.md

## Concepts Used

- Maven Project Structure
- Maven Build Lifecycle
- Exec Maven Plugin

## Implementation

A simple Maven project was created.

The App class contains a main() method which prints a message to the console.

The project was built and executed using Maven commands.

## Commands Used

Compile Project:

mvn compile

Run Project:

mvn exec:java

## Output

Hello Maven Project

## Result

Successfully created and executed a Maven Project using Maven.
