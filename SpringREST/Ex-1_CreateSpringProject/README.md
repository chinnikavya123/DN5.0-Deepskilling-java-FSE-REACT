# Exercise 1 - Create Spring Web Project using Maven

## Objective
Create a basic Spring Boot web project using Maven and verify that the application starts successfully.

## Technologies Used
- Java 24
- Spring Boot 3
- Maven

## Project Structure

```
Exercise1_CreateSpringProject
│── pom.xml
│── README.md
└── src
    └── main
        ├── java
        │   └── com
        │       └── cognizant
        │           └── springrest
        │               └── SpringRestApplication.java
        └── resources
```

## Steps Performed

1. Created a Maven project.
2. Added Spring Boot dependencies.
3. Created the main application class.
4. Ran the application using Maven.
5. Verified that the embedded Tomcat server started successfully.

## Output

```
Tomcat started on port 8080
Spring Boot Project Started
BUILD SUCCESS
```

## Learning Outcome

- Understood the structure of a Spring Boot project.
- Learned how Maven manages dependencies.
- Learned how to run a Spring Boot application.
- Verified successful application startup.
